# Практика

Репозиторий бригады 3 (группа 24ВВВ3) по учебной практике 2 курса.
Вариант: **шейкерная сортировка**.

## Описание задания

В рамках практики необходимо:

1. Установить операционную систему Linux на виртуальную машину (VMware Workstation или другой гипервизор) и описать её работу, состав и процесс установки.
2. Ознакомиться с интегрированной средой разработки (IDE) в ОС Linux и описать работу с ней.
3. Разработать программу на языке **C**, реализующую алгоритм сортировки из варианта, с описанием компиляции и отладки в среде Linux.

Программа реализует ввод данных через `scanf`, арифметические операции, ветвление (`if / else`) и работу с файлами (создание/открытие, чтение, запись, закрытие).

## Алгоритм: шейкерная сортировка

Шейкерная сортировка (cocktail shaker sort) — двунаправленная модификация сортировки пузырьком. Проходы по массиву выполняются поочерёдно в двух направлениях:

1. слева направо — наибольший элемент «всплывает» к концу;
2. справа налево — наименьший элемент «опускается» к началу;
3. границы неотсортированной области сужаются после каждого прохода;
4. сортировка завершается, когда за проход не выполнено ни одной перестановки.

**Характеристики:**

| Параметр | Значение |
|---|---|
| Сложность (худший / средний случай) | `O(n²)` |
| Сложность (лучший случай) | `O(n)` |
| Дополнительная память | `O(1)` (сортировка на месте) |
| Устойчивость | устойчивая (stable) |

## Структура репозитория

- `main` — проверенная рабочая версия программы;
- `dev` — ветка разработки, изменения вносятся сначала сюда, затем переносятся в `main` через Pull Request.
